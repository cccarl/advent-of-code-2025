Doing https://adventofcode.com/2025/ until I get bored or I get skill issued.

## How to run
* Create a folder called `inputs` in root with the puzzle inputs, with the name `day<day number>.txt` depending on the day, for example, day 4 would be `day4.txt`.
* Run with `cargo run <day number>`, for example, day 4 would be `cargo run 4`.
